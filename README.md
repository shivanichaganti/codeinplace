WordGuess Game

A console-based word guessing game developed as the Final Project for Stanford Code in Place 2026 using Python.
Players must guess the secret word one letter at a time before running out of incorrect guesses.

Features:
🎮 Random word selection
🔤 Letter-by-letter guessing
✅ Correct guesses revealed in the word
❌ Incorrect guess counter
✔️ Input validation

Technologies Used:
Python 
Functions
Loops
Conditional statements
String manipulation
